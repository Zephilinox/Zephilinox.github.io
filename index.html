Entity Component System

After reading the blog post on T-Machine I decided to create an Entity Component System by using C++ and SFML. The ECS is unlike that present in Unity, as all components only contain data. Systems operate on groups of components which is how various functioality is implemented, such as movement, controls, and rendering. Each system has access to the EntityManager which in turn stores all Entities as lists of Components, using Variadic Templates to return specific subsets of entities based on the components they have.

  • Date: 2013-02-25
  • Category: Game Development

Galactic Empires

The project Galactic Empires was meant to be a 4x space game made in C++ with SFML, CEGUI, Thor, Lua, and LuaBridge. Development stopped shortly after due to the realisation of the huge scope of such a project and the lack of experience to finish it. Features Lua Integration, CEGUI Integration, State Management, and A* Pathfinding on a Hexagonal Grid.

Improved Chat

I created Improved Chat, the first mod for the MMORPG Trove that modified game logic by using RABCDAsm to dissassemble and reassemble Actionscript files used for the client-side UI of the Chat System by editing raw ASM. This mod made a variety of improvements to the default Trove Chat UI such as clearer colours and an expanded view. It led the way for other modders within the community to create mods with access to the client-side code. At its peak it was in the top 5 mods for Trove with over 2,000 downloads.

  • Date: 2015-08-05
  • Category: Modding
